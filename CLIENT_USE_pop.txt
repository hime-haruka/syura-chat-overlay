[채팅 오버레이 사용 방법]

1. 관리자 페이지에 접속합니다.
   https://syura-chat-overlay.onrender.com/admin/pop

2. 치지직 로그인/인증을 진행합니다.

3. 인증 완료 후 관리자 페이지에서 세션 시작을 누릅니다.

4. OBS를 실행합니다.

5. 브라우저 소스를 추가하고 아래 URL을 입력합니다.
   https://syura-chat-overlay.onrender.com/chat/pop

6. 테스트 메시지 버튼을 눌러 OBS에 표시되는지 확인합니다.

※ Receiver.exe는 사용하지 않습니다.
※ 방송 중에는 Render 서버가 켜져 있어야 합니다.
